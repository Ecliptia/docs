---
title: SearchResult
description: Represents the result of a search query, containing the tracks found, the type of result (playlist, track, search), and potential errors.
icon: lucide:search
---

The `SearchResult` class normalizes the response from the Lavalink `loadTracks` endpoint. It makes it easy to determine if the result was a single track, a playlist, a search result list, or an error.

## Properties

::api-properties{name="Result Fields" description="Data returned from the search operation."}
::api-section{title="Data"}
::
::api-schema{name="loadType" type="LoadType"}
The type of result: `'track'`, `'playlist'`, `'search'`, `'empty'`, or `'error'`.
::
::api-schema{name="tracks" type="Track[]"}
An array of `Track` objects found. Empty if no results or error.
::
::api-schema{name="playlistName" type="string | undefined"}
The name of the playlist (if `loadType` is `'playlist'`).
::
::api-schema{name="exception" type="object | undefined"}
Error details if `loadType` is `'error'`.
::

::api-section{title="Helpers"}
::
::api-schema{name="isPlaylist" type="boolean"}
True if the result is a playlist.
::
::api-schema{name="isTrack" type="boolean"}
True if the result is a single direct track link.
::
::api-schema{name="isSearch" type="boolean"}
True if the result is a list of search results.
::
::api-schema{name="isEmpty" type="boolean"}
True if no tracks were found.
::
::api-schema{name="isError" type="boolean"}
True if an error occurred during loading.
::
::

## Handling Results

::api-card{name="Example Usage"}
#description
How to handle the different load types returned by a search.

#example
```ts
const res = await manager.search({ query: "..." });

switch (res.loadType) {
    case "track":
        player.queue.add(res.tracks[0]);
        break;
        
    case "playlist":
        console.log(`Loaded playlist: ${res.playlistName}`);
        player.queue.add(res.tracks);
        break;
        
    case "search":
        // Add the first result
        player.queue.add(res.tracks[0]);
        break;
        
    case "empty":
        console.log("No matches found.");
        break;
        
    case "error":
        console.error("Load failed:", res.exception?.message);
        break;
}
```
