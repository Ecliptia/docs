---
title: Rest
description: Handles direct HTTP interactions with the Lavalink REST API (v4). Used internally by the Node and Player classes but can be accessed for advanced operations.
icon: lucide:globe-2
---

The `Rest` class is a wrapper around the Lavalink v4 REST API. It handles authentication, request headers, and response parsing. Every `Node` instance has its own `Rest` property.

## Methods

### loadTracks(identifier)
::api-card{name="loadTracks(identifier)" method="loadTracks" returns="Promise<IRESTLoadTracks>" anchorId="loadtracks"}
#description
Resolves a track, playlist, or search query against the Lavalink node.

#default
::api-schema{name="identifier" type="string" required}
The search query or URL (e.g. `ytsearch:faded`, `https://...`).
::

#example
```ts
const res = await node.rest.loadTracks("ytsearch:alan walker");
```
::

### updatePlayer(guildId, data, noReplace)
::api-card{name="updatePlayer(guildId, data, noReplace)" method="updatePlayer" returns="Promise<any>" anchorId="updateplayer"}
#description
Updates the state of a player on the Lavalink server (play track, pause, seek, volume, filters).

#default
::api-schema{name="guildId" type="string" required}
The Guild ID.
::
::api-schema{name="data" type="object" required}
The update payload (track, position, paused, volume, filters, etc.).
::
::api-schema{name="noReplace" type="boolean"}
If true, the operation will be ignored if a track is already playing.
::

#example
```ts
await node.rest.updatePlayer(guildId, {
    paused: true,
    volume: 50
});
```
::

### updateSession(resuming, timeout)
::api-card{name="updateSession(resuming, timeout)" method="updateSession" returns="Promise<any>" anchorId="updatesession"}
#description
Updates the session configuration for resuming.

#default
::api-schema{name="resuming" type="boolean" required}
Whether resuming is enabled.
::
::api-schema{name="timeout" type="number" required}
The number of seconds the session should stay alive after disconnect.
::

#example
```ts
await node.rest.updateSession(true, 60); // 60 seconds
```

### freeFailedAddress(address)
::api-card{name="freeFailedAddress(address)" method="freeFailedAddress" returns="Promise<void>" anchorId="freefailedaddress"}

#default
