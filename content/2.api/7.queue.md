---
title: Queue
description: Manages the list of tracks waiting to be played. Supports adding, removing, shuffling, and manipulating tracks.
icon: lucide:list-music
---

The `Queue` class is a specialized array-like structure that holds the tracks scheduled for playback. It provides helper methods for common music bot operations like shuffling, moving tracks, or clearing the queue.

## Properties

::api-properties{name="Queue Properties" description="State and contents of the track queue."}
::api-section{title="State"}
::
::api-schema{name="size" type="number"}
The number of tracks currently in the queue.
::
::api-schema{name="duration" type="number"}
The total duration of all tracks in the queue (in milliseconds).
::
::api-schema{name="remainingDuration" type="number"}
Duration of the queue + the remaining time of the currently playing track.
::
::api-schema{name="isEmpty" type="boolean"}
True if the queue has no tracks.
::

::api-section{title="Accessors"}
::
::api-schema{name="first" type="Track | undefined"}
The first track in the queue (next to play).
::
::api-schema{name="last" type="Track | undefined"}
The last track in the queue.
::
::api-schema{name="all" type="Track[]"}
Returns a copy of all tracks in the queue array.
::
::

## Methods

### add(track)
::api-card{name="add(track)" method="add" returns="void" anchorId="add"}
#description
Adds one or more tracks to the end of the queue. Respects `maxSize` and `allowDuplicates` settings.

#default
::api-schema{name="track" type="Track | Track[]" required}
The track(s) to add.
::

#example
```ts
// Add a single track
player.queue.add(track);

// Add multiple tracks (e.g. playlist)
player.queue.add(playlist.tracks);
```
::

### remove(index)
::api-card{name="remove(index)" method="remove" returns="Track | undefined" anchorId="remove"}
#description
Removes and returns the track at the specified index.

#default
::api-schema{name="index" type="number"}
The index to remove. Defaults to 0.
::

#example
```ts
const removed = player.queue.remove(2);
console.log(`Removed: ${removed.title}`);
```
::

### clear()
::api-card{name="clear()" method="clear" returns="void" anchorId="clear"}
#description
Removes all tracks from the queue.

#example
```ts
player.queue.clear();
```
::

### shuffle()
::api-card{name="shuffle()" method="shuffle" returns="void" anchorId="shuffle"}
#description
Randomly shuffles the order of tracks in the queue.

#example
```ts
player.queue.shuffle();
```
::

### insert(index, track)
::api-card{name="insert(index, track)" method="insert" returns="void" anchorId="insert"}
#description
Inserts one or more tracks at a specific position in the queue.

#default
::api-schema{name="index" type="number" required}
The index to insert at.
::
::api-schema{name="track" type="Track | Track[]" required}
The track(s) to insert.
::

#example
```ts
// Add "Next Song" to the top of the queue
player.queue.insert(0, nextSong);
```
::

### move(from, to)
::api-card{name="move(from, to)" method="move" returns="boolean" anchorId="move"}
#description
Moves a track from one position to another.

#default
::api-schema{name="from" type="number" required}
The index of the track to move.
::
::api-schema{name="to" type="number" required}
The index where the track should be placed.
::

#example
```ts
// Move the last song to be the next one
player.queue.move(player.queue.size - 1, 0);
```
::

### get(position)
::api-card{name="get(position)" method="get" returns="Track | undefined" anchorId="get"}
#description
Retrieves the track at a specific index without removing it.

#default
::api-schema{name="position" type="number" required}
The index of the track.
::

#example
```ts
const nextSong = player.queue.get(0);
```
::

### has(track)
::api-card{name="has(track)" method="has" returns="boolean" anchorId="has"}
#description
Checks if a specific track object exists in the queue.

#default
::api-schema{name="track" type="Track" required}
The track object to check.
::

#example
```ts
if (player.queue.has(myTrack)) {
    console.log("Track is already queued.");
}
```
::

### removeDuplicates()
::api-card{name="removeDuplicates()" method="removeDuplicates" returns="boolean" anchorId="removeduplicates"}
#description
Removes duplicate tracks from the queue based on their encoded string. Returns `true` if duplicates were found and removed.

#example
```ts
player.queue.removeDuplicates();
```
::

### reverse()
::api-card{name="reverse()" method="reverse" returns="void" anchorId="reverse"}
#description
Reverses the order of the queue.

#example
```ts
player.queue.reverse();
```
::
