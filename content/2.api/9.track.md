---
title: Track
description: Represents a single audio track, containing metadata like title, author, duration, and the encoded string required by Lavalink.
icon: lucide:music
---

The `Track` class wraps the raw data received from Lavalink into a usable object. It provides helper methods for thumbnails, cloning, and serialization.

## Properties

::api-properties{name="Track Fields" description="Metadata and state for the audio track."}
::api-section{title="Core Data"}
::
::api-schema{name="encoded" type="string"}
The base64 encoded string from Lavalink. This is what you send to the node to play the song.
::
::api-schema{name="identifier" type="string"}
The unique identifier of the track (e.g., YouTube Video ID).
::
::api-schema{name="title" type="string"}
The title of the track.
::
::api-schema{name="author" type="string"}
The artist or channel name.
::
::api-schema{name="duration" type="number"}
The length of the track in milliseconds.
::
::api-schema{name="uri" type="string | null"}
The original URL of the track.
::

::api-section{title="State & Extra"}
::
::api-schema{name="position" type="number"}
The starting position of the track (usually 0).
::
::api-schema{name="isStream" type="boolean"}
Whether the track is a live stream.
::
::api-schema{name="isSeekable" type="boolean"}
Whether the track supports seeking.
::
::api-schema{name="sourceName" type="string"}
The source platform (e.g., `youtube`, `spotify`, `soundcloud`).
::
::api-schema{name="artworkUrl" type="string | null"}
The URL of the album art or thumbnail provided by the source.
::
::api-schema{name="requester" type="unknown"}
The user or object that requested this track.
::
::

## Getters

### thumbnail
::api-card{name="thumbnail" type="string | null"}
#description
Returns the artwork URL if available. If not, and the source is YouTube, it automatically generates the `mqdefault` thumbnail URL based on the identifier.

#example
```ts
const thumb = track.thumbnail;
```
::

## Methods

### setRequester(requester)
::api-card{name="setRequester(requester)" method="setRequester" returns="this" anchorId="setrequester"}
#description
Sets the requester for the track. This is useful for displaying who added the song.

#default
::api-schema{name="requester" type="any" required}
The user or object to set as requester.
::

#example
```ts
track.setRequester(message.author);
```
::

### setPosition(position)
::api-card{name="setPosition(position)" method="setPosition" returns="this" anchorId="setposition"}
#description
Sets the starting position of the track. This does NOT seek the active player; it only modifies the track object's metadata for when it is eventually played.

#default
::api-schema{name="position" type="number" required}
Position in milliseconds.
::

#example
```ts
track.setPosition(30000); // Start at 30s
```
::

### clone()
::api-card{name="clone()" method="clone" returns="Track" anchorId="clone"}
#description
Creates a deep copy of the current track.

#example
```ts
const newTrack = track.clone();
```
::

### isPartialTrack()
::api-card{name="isPartialTrack()" method="isPartialTrack" returns="boolean" anchorId="ispartialtrack"}
#description
Returns `true` if the track was created with partial data loading enabled in the Manager options.

#example
```ts
if (track.isPartialTrack()) {
    console.log("This track has incomplete metadata.");
}
```
::

### toJSON()
::api-card{name="toJSON()" method="toJSON" returns="ITrack" anchorId="tojson"}
#description
Converts the Track object back into the raw JSON format expected by Lavalink or for storage.

#example
```ts
const rawData = track.toJSON();
```
::
