This page gets your project installed and ready to run.

## Create a project

```bash
mkdir my-moonlink-bot
cd my-moonlink-bot
npm init -y
```

## Install discord.js

```bash
npm i discord.js
```

## Install Moonlink.js

### Stable release (recommended)

```bash
npm i moonlink.js
```

### Development build (dev)

Use this when you want the latest fixes/features from the repository:

```bash
npm i github:Ecliptia/moonlink.js#dev
```

::alert{type="note" icon="lucide:info"}
The dev build can change more frequently. If youâ€™re building something for production, prefer the stable release.
::
