## Enums

::api-interface{name="NodeState" description="Connection state of a Lavalink node."}
::api-schema{name="CONNECTING" type="0"}
The node is currently attempting to establish a WebSocket connection.
::
::api-schema{name="CONNECTED" type="1"}
The WebSocket connection is open.
::
::api-schema{name="DISCONNECTED" type="2"}
The connection has been closed.
::
::api-schema{name="DESTROYED" type="3"}
The node has been permanently removed and will not reconnect.
::
::api-schema{name="READY" type="4"}
The node has sent the 'ready' payload and is available for use.
::
::api-schema{name="RESUMING" type="5"}
The node is resuming a previous session.
::
::

::api-interface{name="LoadType" description="The type of result returned by a search or track load."}
::api-schema{name="TRACK" type="'track'"}
A single track was loaded.
::
::api-schema{name="PLAYLIST" type="'playlist'"}
A playlist was loaded.
::
::api-schema{name="SEARCH" type="'search'"}
A search result list was returned.
::
::api-schema{name="EMPTY" type="'empty'"}
No matches were found.
::
::api-schema{name="ERROR" type="'error'"}
An error occurred during loading.
::
::

## Types

::api-interface{name="NodeSortStrategy" description="Strategies for selecting the best available node."}
::api-schema{name="leastLoad" type="string"}
(Default) Balances between CPU and player count.
::
::api-schema{name="players" type="string"}
Selects the node with the fewest total players.
::
::api-schema{name="playingPlayers" type="string"}
Selects the node with the fewest active players.
::
::api-schema{name="memory" type="string"}
Selects the node with the most free memory.
::
::api-schema{name="cpuSystem" type="string"}
Selects the node with the lowest system CPU load.
::
::api-schema{name="uptime" type="string"}
Selects the node with the highest uptime.
::
::api-schema{name="random" type="string"}
Selects a node randomly.
::
::api-schema{name="priority" type="string"}
Selects based on the manually assigned priority value.
::
::

::api-interface{name="PlayerLoop" description="Loop modes for the player."}
::api-schema{name="off" type="string"}
No looping.
::
::api-schema{name="track" type="string"}
Repeats the current track indefinitely.
::
::api-schema{name="queue" type="string"}
Repeats the entire queue.
::
::

::api-interface{name="TrackEndReason" description="Reason why a track finished playing."}
::api-schema{name="finished" type="string"}
The track played to completion.
::
::api-schema{name="loadFailed" type="string"}
The track failed to load or crashed.
::
::api-schema{name="stopped" type="string"}
The player was stopped manually.
::
::api-schema{name="replaced" type="string"}
The track was replaced by another track (e.g. `play()` called while playing).
::
::api-schema{name="cleanup" type="string"}
The player was destroyed or cleaned up.
::
::
